REST Service Resources Design

Catalog Resource

Resources URLs XML Repr Meaning
catalog catalog catalog set of all vnfs

Nffgs Resource

Resources URLs XML Repr Meaning
nffgs nffgs nffgs set of all nffgs (without nodes for scalability reasons)
↘ {id} nffgs/{id} nffg single nffg
↘↘ nodes nffgs/{id}/nodes set of all nodes
↘↘↘ {id} nffgs/{id}/nodes/{id} node single node
↘↘↘↘ links nffgs/{id}/nodes/{id}/links set of all links
↘↘↘↘ reachableHosts nffgs/{id}/nodes/{id}/reachableHosts hosts set of all reachable hosts

Hosts Resource

Resources URLs XML Repr Meaning
hosts hosts hosts set of all hosts (without allocated nodes)
↘ {id} hosts/{id} host single host

Connections Resource

Resources URLs XML Repr Meaning
connections connections connections set of all connections
↘ {host1}/{host2} connections/{host1}/{host2} connection single connection

Operations on Resources

This table give a description of designed resources and allowed methods, with relative request and response body and possible status code returned by REST service.
It is based on PDF description of Assignment 3a (design part).

Resource Method Req. Body Status Code Res. Body Meaning
catalog GET 200 OK catalog get the whole VNF collection
500 Internal Server Error server error, try later
nffgs GET 200 OK nffgs get the whole Nffg collection (without nodes)
500 Internal Server Error server error, try later
POST nffg 200 OK nffg create and allocate a complete Nffg
400 Bad Request xml request body malformed
409 Conflict nffg allocation failed
500 Internal Server Error server error, try later
nffgs/{id} GET 200 OK nffg get a complete Nffg
404 Not Found nffg not found
500 Internal Server Error server error, try later
DELETE501 204 No Content undeploy and remove a complete Nffg
404 Not Found nffg not found
500 Internal Server Error server error, try later
nffgs/{id}/nodes POST node 200 OK node create and allocate a Node (without links)
400 Bad Request xml request body malformed
403 Forbidden not applicable on selected resource
409 Conflict node allocation failed
500 Internal Server Error server error, try later
nffgs/{id}/nodes/{id} GET 200 OK node get a complete Node
404 Not Found node not found
500 Internal Server Error server error, try later
DELETE501 204 No Content remove a complete Node
404 Not Found node not found
409 Conflict can't proceed, remove first in/outcoming links
500 Internal Server Error server error, try later
nffgs/{id}/nodes/{id}/links POST link 200 OK link create a Link
400 Bad Request xml request body malformed
403 Forbidden not applicable on selected resource
409 Conflict link already present, try to set overwrite attribute
500 Internal Server Error server error, try later
nffgs/{id}/nodes/{id}/links/{id} DELETE501 204 No Content remove a Link
404 Not Found link not found
500 Internal Server Error server error, try later
nffgs/{id}/nodes/{id}/reachableHosts GET 200 OK hosts get reachable Hosts for seleceted Node
404 Not Found node not found
500 Internal Server Error server error, try later
hosts GET 200 OK hosts get the whole Host collection (without allocated nodes)
500 Internal Server Error server error, try later
hosts/{id} GET 200 OK host get a complete Host
404 Not Found host not found
500 Internal Server Error server error, try later
connections GET 200 OK connections get the whole connections collection
500 Internal Server Error server error, try later
connections/{host1}/{host2} GET 200 OK connection get a specific connection
404 Not Found connection not found
500 Internal Server Error server error, try later
